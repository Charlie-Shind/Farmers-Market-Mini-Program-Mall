## 4. 当前项目阶段

第一阶段目标是把核心交易闭环跑通。

- 登录与身份
- 首页与商品浏览
- 商品详情与搜索
- 购物车与结算
- 支付与订单
- 发货、收货、售后
- 后台审核与运营配置

后续会继续补齐：

- 搜索和推荐
- 消息中心
- 营销活动
- 首页运营位
- 交易风控与续付

### 4.1 运营口径说明

- 当前没有对外开放独立 B 端商家运营入口。
- 平台商品、官方客服入口和默认平台商户统一归属到官方账号。
- 官方账号名称默认使用 `浔源农仓`，可在后台系统配置中修改。
