# 湾源农仓商家端：全页面与设计稿偏差审计报告 (V10 对齐)

本报告针对用户列出的所有页面，逐一审计当前小程序实现与 V10 设计稿（HTML/CSS）之间的样式、布局及功能偏差。

---

## 1. 首页与工作台 (dashboard / workbench)

| 页面名称 | 设计稿规格 | 小程序当前状态 | 样式/布局偏差 | 修正方案/任务 |
| :--- | :--- | :--- | :--- | :--- |
| **首页控制台**<br>`dashboard` | 突出核心数据卡片与今日业绩趋势 | **90% 还原** | 数据卡片折线图（SVG 渲染）与设计稿有微小尺寸比例偏差。 | 统一将 `.data-grid-3` 宽度扩展为 `100%` 并增加 `padding: 0 var(--page-x)`。 |
| **工作台**<br>`workbench` | 整合快捷发布、财务、活动及待办提醒的方格面板 | **90% 还原** | 顶部 `<header>` 导致的高度空白已修复，但下方待办卡片圆角不够圆润，阴影偏重。 | 重置阴影为 `0 8rpx 20rpx rgba(36,53,43,0.04)` 并使用 `!important` 修正 `.action-strip` 边距。 |
