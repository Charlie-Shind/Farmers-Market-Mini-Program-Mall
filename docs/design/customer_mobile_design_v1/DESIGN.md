# 湾源农仓 C 端二级页设计稿规范

## 1. 不改动范围

本设计稿不重做以下页面：

- `pages/index/index` 首页
- `pages/category/category` 分类
- `pages/marketing/marketing` 发现
- `pages/cart/cart` 购物车
- `pages/profile/profile` 我的
- `pages/product/detail/detail` 商品详情

这些页面作为当前 C 端主风格基准，不在本次 HTML 设计稿里替换。

## 2. 本次统一范围

本次只处理 C 端二级页、交易页、资料页、快捷专区页：订单、结算、地址、聊天、收藏、优惠券、积分、溯源、消息详情、快捷专区、个人资料相关页面等。

## 3. 色彩

- 页面背景：`#FAFAF7`
- 分区背景：`#FFFFFF`
- 主色：`#2C4A39`
- 辅助深绿：`#3F6F44`
- 金棕：`#D8A978`
- 橙色强调：`#C65F2D`
- 正文：`#25251F`
- 次级文字：`#8A877F`
- 分割线：`rgba(44,74,57,.10)`

不使用高饱和蓝色系统色，不使用 Ant Design 式状态色。

## 4. 页面结构

普通二级页统一为：

1. 顶部返回栏
2. 搜索 / 标签筛选（按需）
3. 全宽白色分区
4. 商品图、头像、状态、按钮
5. 底部固定操作栏（交易页、表单页按需）

## 5. 组件映射

HTML 设计稿后续转小程序时：

- `.topbar` 映射为 `page-title-bar`
- `.search-pill` 映射为搜索组件
- `.tabs` / `.tab` 映射为状态筛选栏
- `.chip` 映射为筛选标签
- `.goods-row` 映射为商品行
- `.order-card` 映射为订单卡片
- `.cell-list` / `.cell` 映射为设置、地址、表单行
- `.bottom-bar` 映射为底部固定操作栏

## 6. 文案限制

页面可见文案必须中文。组件类名、HTML 注释、文件名可以保留英文，页面上不展示英文说明词。

## 7. 图像规则

订单、商品、收藏、店铺、溯源等页面优先出现商品图或头像。缺少真实图片时使用本地 SVG 占位图，不再用纯文字空块。
