# 当前项目检查结论

检查对象：用户上传的 `farm(1).zip`。

## 1. 构建检查

### backend

初始执行：

```bash
cd backend
npm run build
```

遇到：

```text
nest: Permission denied
```

原因：

```text
node_modules/.bin/nest 没有执行权限。
```

执行：

```bash
chmod +x node_modules/.bin/*
npm run build
```

结果：

```text
backend build 通过。
```

### wxapp

执行：

```bash
cd wxapp
npx tsc --noEmit
```

结果：

```text
TypeScript 检查通过。
```

### admin

执行：

```bash
cd admin
npm run build
```

结果：失败。

原因：

```text
Cannot find module @rollup/rollup-linux-x64-gnu
```

解决：

```bash
cd admin
rm -rf node_modules package-lock.json
npm install
npm run build
```

## 3. 当前最大 P0 问题

`ensureMerchant()` 的身份边界不对。

当前逻辑会优先返回平台官方商家：

```text
system_setting.platformSupportMerchantId 存在 → 所有 merchant 业务都可能使用官方商家
```

这必须改。否则“三身份”无法成立。

正确结果：

```text
普通商家登录 → /merchant/* 只查自己的 Merchant
平台官方商家 → 只用于平台商品、平台客服、平台配置
```
