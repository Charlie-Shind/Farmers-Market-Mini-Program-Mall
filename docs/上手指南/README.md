## 2. 一句话定位

> **浔源农仓**是一个面向农产品交易的电商平台，覆盖 C 端消费者小程序、B 端商家小程序、平台 Web 管理后台三端，当前已完成基础设施与核心交易链路的前后端骨架，正在补齐端到端业务闭环。

## 3. 三端速览

| 端 | 形态 | 目录 | 本地端口 | 启动方式 |
|---|---|---|---|---|
| 后端 | NestJS + Prisma | `backend/` | `6002` | `npm run start:dev` |
| 管理后台 | Vue 3 + Vite | `admin/` | `6007` | `npm run dev` |
| 小程序 | 微信原生 + TS | `wxapp/miniprogram/` | — | 微信开发者工具打开 |

基础设施（PostgreSQL / Redis / MinIO）由仓库根目录的 `docker compose` 拉起，端口见 [02-技术栈与目录结构](./02-技术栈与目录结构.md#四基础设施与本地端口)。

## 4. 第一次跑项目最短路径

1. 启动基础设施
   ```bash
   docker compose -f base/docker-compose.yml --env-file .env.local up -d
   ```
2. 准备后端环境
   ```bash
   cd backend
   cp .env.example .env.local
   # 按需修改密码、密钥、微信小程序 AppID
   npm install
   npm run prisma:generate
   ```
3. 启动后端
   ```bash
   npm run start:dev
   ```
4. 启动管理后台（新终端）
   ```bash
   cd admin
   npm install
   npm run dev
   # 浏览器打开 http://127.0.0.1:6007/admin/
   ```
5. 打开小程序
   - 启动「微信开发者工具」
   - 导入项目，目录指向 `wxapp/`
   - AppID 已写入 `wxapp/project.config.json`

详细步骤、命令解释、常见问题见 [03-本地开发环境搭建](./03-本地开发环境搭建.md)。
