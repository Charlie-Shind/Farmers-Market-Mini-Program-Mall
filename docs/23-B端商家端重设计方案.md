# 23-B端商家端重设计方案

> 现有功能不增减，只改结构和样式。

## 1. 范围

- 工作台
- 商品
- 订单
- 财务
- 提现
- 营销
- 店铺

不加：

- 成长
- 课程
- 大盘
- 预警
- 新营销

## 2. 顺序

1. 工作台
2. 商品
3. 订单
4. 财务
5. 店铺
6. 营销
7. 提现

## 3. 页面

### 工作台

- 店铺信息
- 店铺状态
- 核心数字
- 常用入口
- 待办
- 订单概览
- 收益概览

### 商品

- 商品总数
- 上架中
- 已下架
- 待审核
- 搜索
- 筛选
- 列表
- 新增

### 订单

- 待接单
- 待发货
- 已发货
- 售后中
- 搜索
- 列表
- 详情

### 财务

- 本周收入
- 可提现金额
- 冻结金额
- 流水

### 提现

- 可提现余额
- 提现金额
- 收款账户
- 备注
- 记录

### 营销

- 活动入口
- 活动列表
- 状态
- 编辑
- 删除

### 店铺

- 店铺信息
- 联系人
- 审核状态
- 资质
- 退出登录

## 4. 样式

- 主色：农场绿
- 强调色：橙色
- 辅助色：蓝色
- 背景：浅米白
- 卡片：大圆角
- 阴影：轻
- 标题重，说明轻

## 5. 交互

- 先状态，后内容
- 先待办，后列表
- 空态给现有入口
- 按钮固定清楚
- 重要状态明显

## 6. 验收

- 不加功能
- 不改路由
- 入口能找到
- 页面整齐
- 重点突出
- 风格统一
