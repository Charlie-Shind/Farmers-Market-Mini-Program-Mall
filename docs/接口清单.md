# 后端接口清单基线

> 末轮更新：2026-06-17
> 后端 184 端点（注册方法 185，含 1 个 legacy 重复 `POST /auth/refresh` 待清理），0 个 scaffold 端点

## 说明

- 服务统一前缀：`/api`
- 鉴权方式说明：
  - `Public`：无需登录
  - `User/Guest`：普通用户或游客 token
  - `User`：普通用户 token
  - `Merchant/User`：商户或普通用户 token
  - `Admin`：后台管理员 token
  - `Optional`：可带可不带 token，带 token 时返回个性化信息
- 资源 ID 风格：商品/商户/用户用 `id`、订单/退款/提现/团长分佣等用业务号 `orderNo` / `refundNo` / `applyNo` 等
- 鉴权装饰器：`@Public()` 公开、`@Roles('USER'|'GUEST'|'MERCHANT'|'ADMIN')` 角色限定、缺省 `@CurrentUser()` 需登录
- 无 WebSocket 网关、无 Cron 定时任务

## 店铺主页（公开浏览）

> 2026-06-17 新增：方便 C 端 user/游客 公开查询已入驻店铺及其在售商品。`MerchantPublicController` 在 `/api/app/merchants` 下，鉴权 Public。仅返回 `merchant.status=1`（已通过）的店铺及其 `product.status=1` 且 `product.auditStatus=2` 的商品。`contactMobile` 等敏感字段不在公开端点回传。

| 方法 | 路径 | 鉴权 | 说明 |
| --- | --- | --- | --- |
