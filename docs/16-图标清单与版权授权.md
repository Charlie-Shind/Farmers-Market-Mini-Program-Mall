## 4. 建议的申请顺序

1. **第一步**：在 **iconfont.cn** 注册账号并创建项目「浔源农仓」
2. **第二步**：从 **Lucide** 导出上述全部图标（SVG 格式），按场景打 tag
3. **第三步**：上传到 iconfont.cn 项目里，生成 CSS/JS 链接
4. **第四步**：补充少量 Lucide 没有的"中国风/助农/特产"类图标（可请设计同事绘制或去 iconfont 搜）
5. **第五步**：导出字体包，纳入 `wxapp/miniprogram/assets/icons/` 和 admin 端
